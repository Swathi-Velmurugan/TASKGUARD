TaskGuard is a role-based project task management system built for teams that need clear ownership, capacity tracking, and accountability from portfolio down to a single logged hour.

Get Started

What is TaskGuard?

TaskGuard gives organizations a single place to plan project initiatives, break them down into milestones and tasks, assign work to contributors, and review submitted hours before they count toward delivery. Every action is protected behind secure, JWT-based authentication and enforced by role-specific permissions.

Whether you're a Director overseeing budgets across a portfolio, a Manager coordinating a delivery schedule, or a Contributor logging your daily work, TaskGuard shows you exactly what you need and nothing you don't.

Three roles, one system
Project Director

Owns initiatives, budgets, and system accounts.

Project Manager

Plans milestones, creates tasks, and assigns work.

Team Contributor

Updates task status and logs completed hours.

What you can do

Project Initiatives

Create portfolio-level initiatives with allocated budgets, timelines, and real-time budget consumption tracking.

Milestones

Break initiatives into delivery phases with target dates and allocated hour budgets.

Tasks & Assignment

Assign work to contributors while the system enforces a 40-hour weekly capacity limit automatically.

Work Submissions

Contributors log hours and notes; managers approve or reject each submission with feedback.

Account Management

Directors provision, edit, suspend, or remove system accounts across all three roles.

Secure by Design

Every request is authenticated with a signed JWT and checked against role-based access rules before it runs.
